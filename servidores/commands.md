# Comandos Linux

## Comprimir

```
zip -r nombre_comprimido.zip directorio_a_comprimir
```

## Descromprimir

```
unzip srcpython20190610.zip
```

## Descargar archivos

```
scp usuario@host:directorio/ArchivoOrigen ArchivoDestino
```

## Subir archivos

```
scp ArchivoOrigen usuario@host:directorio/ArchivoDestino
```
